Heat Day Calculator - Version 1.0 6/14/16
Web App Development Final Project
--------------------------------
Creators: Sharon Liu, Sophie Zhao
--------------------------------
GENERAL USAGE
---------------------------
- User can input:
	- a desired city and state
	- The number of school days missed so far in the school year
	- Education budget percentage
	- The willingness of the powers that be (the school board) to cancel school
		-(e.g. LCPS is a 10, FCPS is a 1)
- App takes data into Weather Underground API and takes values into an algorithm to calculate percentage
-Weather and temperature along with percentage of a heat day outputted to user
